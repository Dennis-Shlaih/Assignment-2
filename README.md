# Assignment 2: Expense Tracker SPA

A single-page expense tracking application built with HTML, CSS, and JavaScript.

## Features

- Add expenses with description, amount, category, and date
- Form validation with error messages
- Delete expenses
- Filter expenses by category
- Sort expenses by date or amount
- Running total of expenses
- Category totals
- Expense count
- Local storage persistence
- Currency conversion from USD to EUR using a public exchange rate API
- Loading and error states for API requests
- Responsive design

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

1. Clone the repository.
2. Open `index.html` in a web browser.
